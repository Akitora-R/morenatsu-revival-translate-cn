# RPY 标签 → KS 翻译文件映射文档

> 本文档列出每个 RPY 文件中的标签前缀，以及对应的旧版 KS 翻译文件。
> 用于快速定位：某个 RPY 标签的翻译结果在哪个 KS 文件中。

## 使用说明

- **标签前缀**：RPY 翻译条目的 ID 去掉末尾哈希部分（如 `shun16` 来自 `shun16_c00615f0`）
- **对应 KS 文件**：该标签的翻译可在 `output/texts/ks_texts/` 目录下找到对应的 `.jsonl` 文件
- **条目数**：该标签前缀下有多少条未翻译文本
- **未找到匹配**：表示该标签是 Ren'Py 新增内容，旧版 KS 中不存在对应翻译
