## 翻译规则

1. **优先使用旧版翻译**：所有翻译严格以 `extract/` 下的 `.ks` 文件为准
2. **新内容机翻**：旧版中不存在的台词，结合上下文进行机翻
3. **注释标记**：每条翻译上方添加 `# [旧版翻译]` 或 `# [机翻-新内容]` 注释
4. **占位符保留**：Ren'Py 变量如 `[fn]`、`[ln]`、`[漏れ]` 等保持原样
